Convert and Send Form






API Description

The /api/v1/convert-and-send endpoint allows you to convert a document from a given URL to a specified format. The converted document will be returned as a downloadable file.

Request

URL: https://services.samoliot.ru/api/v1/convert-and-send

Method: POST

Headers:

  • Content-Type: application/json
  • Authorization: Bearer {token}

Body:

{
    "url": "http://example.com/document",
    "format": "pdf",
    "clip_to_element": "element_id (optional)",
    "emulate_media_type": "emulate_media_type (optional)"
}

Response

Status Code: 200 OK

Headers:

  • Content-Disposition: attachment; filename="converted_file.pdf"
  • Content-Type: application/pdf

Body: The converted file in the specified format.
